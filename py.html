Python递归函数深度解析

往事随风 2025-05-06 2567 阅读

什么是递归函数？

递归函数是指在函数定义中调用自身的方法。一个典型的递归实现包含两个要素：

def factorial(n):
    if n == 1:
        return 1
    else:
        return n * factorial(n-1)

递归的三大要素

1. 基线条件（Base Case）

2. 递归条件（Recursive Case）

# 斐波那契数列示例
def fibonacci(n):
    if n <= 1:   # 基线条件
        return n
    else:        # 递归条件
        return fibonacci(n-1) + fibonacci(n-2)

递归优化技巧

使用缓存装饰器提升性能：

from functools import lru_cache

@lru_cache(maxsize=None)
def fib(n):
    if n < 2:
        return n
    return fib(n-1) + fib(n-2)
小贴士：递归深度默认限制为1000层，可通过sys.setrecursionlimit()调整
